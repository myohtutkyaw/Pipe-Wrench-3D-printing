The pipe wrench is a widely used industrial tool with a long history, originally invented in 1869 by Daniel C. Stillson and later improved by Johan Petter Johansson in 1888. It features:

One fixed jaw and one movable jaw

Serrated teeth for high grip

A mechanism allowing rotation in one direction without slipping

Most common sizes range from 150 mm to 1200 mm.
This project focuses on producing a compact 120 mm prototype via 3D printing.
